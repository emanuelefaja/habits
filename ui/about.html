Our technology stack is carefully chosen for reliability and simplicity:

  • Backend:
    • Go for a fast, scalable and reliable backend
    • Session management with alexedwards/scs
    • SQLite integration via mattn/go-sqlite3
    • Secure password hashing with golang.org/x/crypto
    • Environment variables with godotenv
    • goldmark for markdown processing
    • Live reload development using Air
    • gopkg.in/yaml.v3 for YAML parsing
  • Frontend:
    • AlpineJS for lightweight interactivity
    • Tailwind CSS for modern, responsive styling
    • Chart.js for data visualization
    • SortableJS for drag-and-drop functionality
    • Emoji Mart for emoji picker and search
    • Google Fonts for modern typography (Inter)
    • Canvas Confetti for celebration animations
    • PinesUI for Alpine & Tailwind UI component library
  • Database:
    • SQLite for reliable, zero-dependency data storage
    • Litestream for continuous SQLite replication
  • Infrastructure:
    • Cloudflare for DNS and DDoS protection
    • Render for reliable cloud & database hosting.
    • Github for version control
